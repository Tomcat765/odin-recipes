Spaghetti

Description

Follow this easy recipe in order to create the finest spaghetti you have ever tasted

Before creating this masterpiece, please be aware that you will need to have correct ingredients for it to be perfect

To be sure of that, please follow the ingredients list below

Ingredients

  • Minced meat
  • Flour
  • Salt
  • Origano
  • Onions
  • Parmezzano

Steps

  1. Cook the onions and meat
  2. Add origano
  3. Add salt
  4. Add flour
  5. Mix everything up
